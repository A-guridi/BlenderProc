  • max_b2 – List of maximum bounding box points for b2.
  • used_check – The operation used inside of the is_overlapping1D. With that it possible to change the collision check from volume and surface check to pure surface or volume checks.

src.utility.BlenderUtility.check_intersection(obj1, obj2, skip_inside_check=False, bvh_cache=None)[source]¶

Checks if the two objects are intersecting.

This will use BVH trees to check whether the objects are overlapping.

It is further also checked if one object is completely inside the other. This check requires that both objects are watertight, have correct normals and are coherent. If this is not the case it can be disabled via the parameter skip_inside_check.

Parameters:
  • obj1 – object 1 to check for intersection, must be a mesh
  • obj2 – object 2 to check for intersection, must be a mesh
  • skip_inside_check – Disables checking whether one object is completely inside the other.
Returns:

True, if they are intersecting
